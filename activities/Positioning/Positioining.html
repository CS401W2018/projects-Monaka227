Answers

  1. Block tags take up the full width of their container and start on a new line, while inline tags take up only as much width as necessary and remain within the flow of text.

    Examples: Block tag: <div>, Inline tag: <span>.

  2. Normal flow refers to the default layout behavior of elements in HTML, where block elements stack vertically and inline elements stack horizontally within the flow of content.
  3. The float property allows elements to float to the left or right of their container, causing other content to wrap around it. Clearing the float (clear) prevents elements from wrapping around the floated elements and returns to normal flow.
    • absolute: Positions the element relative to its nearest positioned ancestor.
    • fixed: Positions the element relative to the viewport (stays in place while scrolling).
    • relative: Positions the element relative to its normal position in the flow.
  5. Use CSS to make buttons responsive, such as using media queries to adjust size, padding, and layout for mobile screens. You can use flexbox or grid layout to center buttons and adjust their appearance for touch screens.
