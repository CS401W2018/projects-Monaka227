1.Link to an outside website with an absolute link.
2.Explain the difference between an absolute link and a relative link. Why would you use relative links?
Absolute link - Use when accessing an external site or when the domain is different. It always points to the exact location, so it can be used even if the linked URL is on a different server or domain; since it describes the entire URL, it is not affected by changes in the file structure
Relative link - Often used within the same site. If the site is moved to a different server or the domain changes, there is no need to modify the link.
Why use relative links? - Relative links are especially useful when referring to pages or files within the same domain.
3.Link to another engagement assignment (something in an activity folder).
4.ist the 3 rules with relative links.

  1. Same Directory Rule - When linking to a file in the same directory as the current HTML file, simply use the file name.
  2. Parent Directory Rule - To link to a file in a parent directory (one level up), use ../ followed by the file name or folder path.
  3. Subdirectory Rule - When linking to a file in a subdirectory (a folder inside the current directory), include the folder name followed by the file name.
