  1. AJAX (Asynchronous JavaScript and XML) is a technique used in web development to create dynamic, interactive web pages. It allows a web page to send and receive data from a server asynchronously, without reloading the entire page. This improves the user experience by enabling faster updates and smoother interactions.
    • Code Reusability: A separate JavaScript file can be reused across multiple HTML files, reducing duplication.
    • Improved Readability: Keeping JavaScript code separate makes the HTML and JavaScript easier to read and maintain.
    • Caching: Browsers can cache the JavaScript file, improving load times for subsequent visits.
    • Separation of Concerns: It follows the best practice of keeping structure (HTML), styling (CSS), and behavior (JavaScript) separate, leading to cleaner and more modular code.
