1 Internet

The Internet is a vast global communication network that connects computers and other devices worldwide. It enables data to be transmitted using various protocols.

2 Web

The Web is a system for sharing documents and information over the Internet. It provides an interface for viewing web pages written in HTML through a web browser.

3 How are the Internet and Web different

The Internet is the physical network, connecting computers, servers, and routers globally. The Web is a service or application that operates on the Internet. It refers to websites and content viewed through web browsers.

4 ASCII and Unicode

ASCII is a standard code that converts English letters, digits, and symbols into binary data using 7 bits. It was primarily used to represent English text. Unicode is a character encoding standard designed to represent all characters used in various languages worldwide. Unlike ASCII, which only supports English characters, Unicode supports a wide range of languages, including Japanese, Chinese, and Arabic.

5 What is UTF-8

UTF-8 is one of the encoding formats used to represent Unicode characters efficiently. It uses variable-length encoding (1 to 4 bytes) and is compatible with ASCII. It is the most commonly used encoding format on the Web.

6 URL

A URL is a string that specifies the location of a resource (such as a web page or image) on the Web. It typically includes a domain name and protocol.

7 Domain Name

A domain name is a human-readable address that corresponds to an IP address on the Internet.

8 HTTP

HTTP is the protocol used to transfer data between a web browser and a server. It is commonly used to retrieve web pages, images, and other online content.

9 HTTPS

HTTPS is the secure version of HTTP.

10 IP Adress

An IP address is a unique number used to identify a device on the Internet.

11 DNS

DNS is a system that translates domain names into corresponding IP addresses.

12 W3C Recommendations

W3C (World Wide Web Consortium) is an organization that develops web standards. Its recommendations, such as those for HTML and CSS, serve as guidelines for web developers and companies to follow when creating web content.

13 XML, XHTML, HTML, HTML5

XML (Extensible Markup Language), XHTML (Extensible HyperText Markup Language), and HTML (HyperText Markup Language) are all markup languages used to structure and present data on the web.
