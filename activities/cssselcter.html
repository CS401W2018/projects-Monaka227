CSS Selectors

Questions

  1. List out the different ways to select an item.
  2. What is the difference between an ID and a Class?
  3. What is the difference between a SPAN tag and a DIV tag?
  4. What is the difference between a DIV tag and HEADER, MAIN, FOOTER?
  5. What are different ways to select a color?

Answers

    1. Tag Name
      ID
      class
      Children

    2. ID: for a unique ID for each element; an ID can only be used once on a page.
      Class: for styles that can be applied to multiple elements; the same class name can be used for multiple elements.

    3. SPAN tag: This is an inline element. Used primarily to style a portion of text or a small section of text. Used to embellish a portion of text without altering the flow of the block.
      DIV tag: This is a block-level element. Used to organize large groups of sections or content and can contain other block elements.

    4. DIV tags: These are generic container elements, have no specific meaning, and are used to group sections within a page.
      HEADER, MAIN, and FOOTER: These are semantic elements and have specific roles.

    5. Color Name
      RGB
      RGBA
      HEX code
      HSL
      HSLA
