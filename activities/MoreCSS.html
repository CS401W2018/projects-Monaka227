More CSS

Questions

  1. What is the difference between block tags and inline tags.
  2. What is the margin, border, and padding?
  3. What are units of measurement you can use in CSS?
  4. When would you use a figure tag?

Answers

  1. Block tags create elements that take up the full width of their container, stacking vertically. Inline tags, on the other hand, only take up as much space as their content requires and stack horizontally within the same line.
  2. Margin is the space outside the element's border, separating it from other elements. Border is the line surrounding the element. Padding is the space between the content of the element and its border.
  3. CSS uses units like pixels (px), em, rem, percentages (%), and viewport units like vw (viewport width) and vh (viewport height).
  4. The figure tag is used to wrap content such as images, diagrams, or illustrations that are referenced within the main content, often along with a caption.
