CSS Basics

Questions

  1. Explain the difference between inline, embedded, and external CSS.
  2. Explain why you should only use external CSS.
  3. What is a selector used for?

Answers

    • Inline CSS: Applied directly to an HTML element using the style attribute.
    • Embedded (Internal) CSS: Written inside the style tag in the head section of an HTML document.
    • External CSS: Written in a separate .css file and linked to the HTML document using a link tag.
  2. Using external CSS because it keeps HTML and CSS separate, making the code easier to maintain and reuse across multiple pages. It also improves performance since the stylesheet can be cached by browsers, reducing load times. External CSS is especially useful for larger, more scalable websites.
  3. A CSS selector is used to target and apply styles to specific HTML elements. It defines which elements on a webpage will be affected by the CSS rules.
