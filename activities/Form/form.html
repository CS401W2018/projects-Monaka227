  1. action attribute: Specifies the URL to send the form data to when it is submitted. This is usually the URL of the server-side script that will process the data (e.g., action="submit_form.php").
  2. method attribute: Specifies the HTTP method to use when sending form data. The two most common methods are "GET" and "POST".
    • GET: Sends data appended to the URL. Because the data is visible in the URL, it’s suitable for small amounts of data, like search queries.
    • POST: Sends data in the request body, not visible in the URL, making it more secure. It’s ideal for submitting form data and large amounts of information.
